# CNBC TV18 — UX Case Study

A markets-first redesign for a brand about to pivot.

The web app was a research terminal. The mobile app was a news reader. Investors used the first one — and CNBC's 2026 brand refresh meant the gap had to close. This case study walks through the research, the reframe, the four design pillars, the iteration arc, and what shipped vs. what queued.

**Live:** [open `index.html`](./index.html) in any browser.
